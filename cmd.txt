1. Calculate LD scores and estimate heritability:
```bash
python3 ldsc.py \
-M all \
-s ./data/full.sumstats \
-r ./data/eur_w_ld_chr/ \
-o ./results/test
```

2. Calculate LD scores only:
```bash
python3 ldsc.py \
-M ldsc \
-s ./data/full.sumstats \
-r ./data/eur_w_ld_chr/ \
-o ./results/test
```

3. Estimate heritability only:
```bash
python3 ldsc.py \
-M h2 \
-s ./results/test.txt \
-n 61220 \
-o ./results/test
```
